h2. The BrowserPlus Platform

This is the home of the BrowserPlus platform implementation: portable client
code that plugs into all popular browsers to provide a new, higher level
plugin API.  A Plugin API optimized for secure and portable scriptable
web plugins.

_BrowserPlus is an open source project created and supported by Yahoo!_.
